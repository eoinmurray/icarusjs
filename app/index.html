Introduction

Icarus.js fully simulates the Hanbury Brown Twiss experiment for the Bell input states.

These state are given in bra-ket notation as $$\left|\psi\right\rangle = \frac{1}{\sqrt{2}} \left(\left|\uparrow\uparrow\right\rangle + \left|\downarrow\downarrow\right\rangle \right)$$ $$\left|\psi\right\rangle = \frac{1}{\sqrt{2}} \left(\left|\uparrow\uparrow\right\rangle - \left|\downarrow\downarrow\right\rangle \right)$$ $$\left|\psi\right\rangle = \frac{1}{\sqrt{2}} \left(\left|\uparrow\downarrow\right\rangle + \left|\downarrow\uparrow\right\rangle \right)$$ $$\left|\psi\right\rangle = \frac{1}{\sqrt{2}} \left(\left|\uparrow\downarrow\right\rangle - \left|\downarrow\uparrow\right\rangle \right)$$
